++ Description ++

This plugin adds the Facebook Comments Plugin box to your website and sync it with your Wordpress defult comment system. With this plugin you are able show Facebook comments on your website for SEO purposes

- Features:
+ Display Facebook Comments Plugin box in your single page.
+ Update/delete comments instantly when new FB comment added or delete.
+ Feature to sync all current FB comments to WP built-in comments.
